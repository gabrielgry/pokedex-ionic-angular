## Como executar o projeto

1. Clone o repositório

    ```bash
    git clone https://github.com/gabrielgry/pokedex-ionic-angular.git
    ```

2. Instale as dependências

    ```bash
    cd pokeapp
    npm install
    ```

3. Execute o aplicativo

    ```bash
    ionic serve
    ```
